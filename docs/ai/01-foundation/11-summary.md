# 11.3 From Text to Intelligence

When a user sends a prompt, the model does not read English, Vietnamese, or any other human language directly.

Instead, the request goes through multiple stages.

```text
User

↓

Text

↓

Tokenizer

↓

Token IDs

↓

Embeddings

↓

Transformer

↓

Logits

↓

Sampling

↓

Next Token

↓

Response
```

Every response generated by an LLM follows this pipeline.

# 11.5 How Meaning Emerges

One of the most surprising discoveries in modern AI is that sophisticated language abilities emerge from a relatively simple objective.

```text
Predict

↓

Next Token

↓

Repeat Trillions of Times

↓

Learn Grammar

↓

Learn Facts

↓

Learn Programming

↓

Learn Reasoning

↓

Learn Language
```

The model is never explicitly taught:

- English grammar
- Python syntax
- Software architecture
- Mathematical reasoning

Instead, these capabilities emerge from large-scale training.

# 11.6 The Role of Each Component

Every component in an LLM has a specific responsibility.

| Component | Responsibility |
|-----------|----------------|
| Tokenizer | Converts text into token IDs |
| Embedding Layer | Converts token IDs into vectors |
| Positional Encoding | Preserves token order |
| Transformer | Learns contextual representations |
| Attention | Determines relevant information |
| Feed Forward Network | Learns nonlinear transformations |
| Softmax | Converts logits into probabilities |
| Sampling | Selects the next token |

Removing any component would significantly reduce model performance.

---

# 11.7 The Role of Attention

Attention is the core innovation of modern AI.

Instead of processing text sequentially, every token dynamically determines which other tokens are important.

```text
Sentence

↓

Attention

↓

Relevant Tokens

↓

Contextual Representation
```

This mechanism enables:

- Long-range reasoning
- Better language understanding
- Parallel computation

Without attention, today's LLMs would not exist.

# 11.8 Why Embeddings Matter

The Transformer never processes words directly.

It processes vectors.

```text
Text

↓

Token IDs

↓

Embeddings

↓

Vectors

↓

Transformer
```

Embeddings preserve semantic similarity.

For example:

```text
Cat

Dog

Tiger

Lion
```

are located close together in embedding space.

This principle later becomes the foundation of semantic search and Retrieval-Augmented Generation.

# 11.9 Why Context Matters

An LLM can only reason about information currently inside its Context Window.

```text
Prompt

↓

Context Window

↓

Inference

↓

Response
```

Information outside the context window simply does not exist from the model's perspective.

This limitation motivates later topics such as:

- RAG
- Memory Systems
- Context Engineering

---

# 11.10 Why LLMs Hallucinate

LLMs do not retrieve facts from a database.

Instead, they predict statistically likely tokens.

```text
Prompt

↓

Probability Distribution

↓

Next Token

↓

Repeat
```

Consequently, they may generate responses that sound convincing but are factually incorrect.

This phenomenon is called hallucination.

It is one of the primary motivations for Retrieval-Augmented Generation.

---

# 11.11 What an LLM Is Not

Understanding what an LLM cannot do is just as important as understanding what it can.

An LLM is **not**:

- A database
- A search engine
- A reasoning engine with perfect logic
- A persistent memory system
- A source of guaranteed truth

Instead, it is:

> A probabilistic language model trained to predict the next token.

Everything else emerges from this objective.

# 11.15 Common Misconceptions

The following misconceptions are frequently encountered among beginners.

### "LLMs understand language like humans."

False.

They learn statistical patterns from large datasets.

---

### "More parameters always mean better results."

False.

Data quality, alignment, and architecture also matter.

---

### "A larger context window eliminates hallucinations."

False.

Hallucinations can still occur even with very large contexts.

---

### "Prompt Engineering is the most important AI skill."

Not anymore.

Modern production systems rely more heavily on:

- Context Engineering
- Retrieval
- Memory
- Agent Architecture

---

### "LLMs remember every conversation."

False.

Persistent memory must be implemented outside the model.
